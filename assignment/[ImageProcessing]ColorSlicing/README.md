## Color Slicing

1. 얼굴사진을 촬영하고 color slicing을 통해서 살짝 영역 추출
    -  본인 혹은 동양인 얼굴을 가지고 수행할 것
2. 추출된 영역과 추출되지 않은 영역에 대한 정보를 matrix 형태로 map 보관해야 함
3. 추출된 영상에 대해서, smoothing filter 적용
4. 위의 map을 이용하여, 얼굴 영역에 대해서 smoothing된 영상을 overwrite
5. 결과 분석 및 고찰
