# InputString

문자열을 사용자로부터 계속적으로 입력 받아 입력 받은 문자열, 문자열의 길이, 입력된 문자열의 개수, 문자의 총합을 출력하는 프로그램을 문자 배열을 이용하여 작성하고 수행결과를 보이시오.

**수행 예 :**

```
문자열 입력: This
문자열 입력: is
문자열 입력: example
문자열 입력: for
문자열 입력: you
문자열 입력: 0 (0이 입력되면 수행을 끝냄)
```

**결과 :**

```
This 4
is 2
example 7
for 3
you 3
문자열의 수: 5
문자의 수: 19
```
