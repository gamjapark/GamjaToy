# Score Processing

2018 2학기 광운대학교 컴퓨터 소프트웨어학과

고급 C 프로그래밍 및 설계

10주차 실습1

## 점수 처리 프로그램

키보드로부터 학생 3명이 4과목 점수를 차례대로 입력받아 2차원 배열에 저장하고, 각 학생이 평균 점수를 출력하는 프로그램을 작성하시오.

1. 2차원 배열 정의
   - 학생 3명의 4과목 점수 입력할 수 있는 배열
2. 키보드로부터 차례대로 점수 입력
3. 학생 당 평균 점수 계산 및 출력
